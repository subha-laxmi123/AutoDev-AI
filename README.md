# 🤖 AutoDev AI

AutoDev AI is a multi-agent code review assistant built using Python, Streamlit, and the Gemini API. It helps developers analyze Python code, identify bugs, suggest improvements, and improve code quality with AI.

## Features
- Upload Python (.py) files
- AI-powered code analysis
- Detects bugs and code quality issues
- Suggests improvements and best practices
- Built with a multi-agent architecture

## Tech Stack
- Python
- Streamlit
- Gemini API
- python-dotenv

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
